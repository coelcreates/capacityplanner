      üîë Getting Your Asana Personal Access Token

      1. Go to Asana Developer Console
      2. Click "Create new personal access token"
      3. Give it a name (e.g., "Capacity Planner")
      4. Copy the generated token (you'll only see it once!)
      5. Press Ctrl+Shift+A to open the admin panel
      6. Paste your token in the "Asana Personal Token" field

      üìã Finding Your Project ID

      1. Open your Asana project in the browser
      2. Look at the URL: https://app.asana.com/0/PROJECT_ID/list
      3. Copy the PROJECT_ID number from the URL
      4. In the admin panel, paste it in the "Default Asana Project ID" field
